OSI & TCP/IP Model :

OSI is Open System Information model. It's a 7 layer model. 
1. Application- It's for user level services > Browser, email, ssh
2. Presentation - Data format Conversation , Encryption , Compression
3. Session - Manages connection between two systems > login sessions, SSH sessions
4. Transport - Responsible for end to end communication and reliability
5. Network - It handles IP addressing and routing between networks
6. Data Link- It Communicate with Same networks > MAC address , Switch
7. Physical - Actual hardware like cables, hub

TCP/IP is basically a modified model of OSI Model . It has 4 layer .

1. Application
2. Transport
3. Internet
4. Network Address

IP Address : Internet Protocol address is an unique identity of a device in a network.

Two Types of ID address :

IPv4: 192.168.X.X
IPv6: 2001:0db8::1

Class A > For Large network
Class B > Medium Network
Class C > Small Network
